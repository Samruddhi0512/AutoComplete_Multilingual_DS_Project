# AutoComplete_Multilingual_DS_Project

## Project Description
This project implements an autocomplete system using Data Structures.
It provides real-time word suggestions based on user input and supports multiple languages.

The system is designed using Trie data structure for fast prefix searching and efficient suggestion generation.

## Features
- Autocomplete suggestions
- Multilingual support
- Fast search using Trie
- Frequency-based ranking
- Recently used words priority

## Data Structures Used
- Trie (Prefix Tree)
- Hash Map
- Vector / List

## Technology Used
- Language: C++
- Concepts: Data Structures, OOP, File Handling

## How It Works
1. User enters a prefix
2. Trie searches matching words
3. Suggestions are ranked
4. Top results are displayed

## How to Run
Compile and run the program using:
g++ main.cpp -o autocomplete
./autocomplete
